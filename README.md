#Pocket World

## Pocket world is a social website where you can:

- 🖊 Post your status 
- 👍 Like and dislike a status 👎
- 🔎 Search for friends
- ➕ Add a friend
- 🚫 Block a friend
- ❌ Delete a friend
- 🌎 Check people you may know
<br />

### Languages:
- Html & CSS
- JavaScrip
- PHP
- SQL

### Tools:
